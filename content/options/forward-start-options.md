# Forward Start Option 是什么？

Forward Start Option 是未来才开始生效的期权，执行价通常在未来某个时间根据标的价格设定。

它常用于员工激励、结构化产品和远期波动率交易。

新手需要看：

1. 期权什么时候开始。
2. 执行价如何确定。
3. 到期日和结算方式。
4. 波动率假设是否合理。

它的重点不是当前方向，而是未来某段时间的波动和定价。不要用普通现货期权直觉直接套用。
