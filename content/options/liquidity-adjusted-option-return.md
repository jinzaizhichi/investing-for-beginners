# 期权收益要怎样扣除流动性成本？

期权账面收益如果没有扣除买卖价差和滑点，容易高估策略质量。

评估一笔交易时要扣：

1. 开仓价差成本。
2. 平仓价差成本。
3. 多腿组合的净价损耗。
4. 成交等待导致的追价成本。
5. 大仓位冲击成本。

低价期权的价差占比可能很高。看起来赚 20%，实际可能大部分被进出成本吃掉。

复盘时记录理论中间价、实际成交价和可成交规模，才能判断策略是真的有效，还是只在纸面上有效。
