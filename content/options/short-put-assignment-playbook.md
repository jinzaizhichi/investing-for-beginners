# 卖 Put 被指派后怎么办？

卖 Put 被指派意味着你需要按执行价买入股票。指派不是意外灾难，而是卖 Put 前就应该计划好的结果。

处理步骤：

1. 确认买入股票数量和成本。
2. 计算有效成本：执行价减去收到的权利金。
3. 重新评估是否仍愿意持有。
4. 如果不愿持有，制定卖出或止损计划。
5. 如果愿意持有，纳入股票仓位管理。

卖 Put 前就要问：如果明天被迫买 100 股，我开心吗？如果答案是否定，就不该开仓。
