# Christmas Tree Spread 是什么？

Christmas Tree Spread 是一种多腿期权价差，通常通过不同行权价组合表达有限方向观点。它比普通垂直价差更复杂。

新手重点看：

1. 最大收益区间在哪里。
2. 最大亏损是否有限。
3. 哪些腿是买入，哪些腿是卖出。
4. 到期前流动性是否足够。
5. 多腿成交成本是否过高。

它名字有趣，但不是入门策略。看不清盈亏图和指派风险，就不要为了低成本贸然开仓。
