# Jensen's Alpha 说明什么？

Jensen's Alpha 衡量一个组合在考虑市场风险后，是否获得了超额收益。它常用 CAPM 作为基准。

如果 Alpha 为正，表示组合收益高于按 Beta 应该获得的收益；如果为负，说明承担风险后表现不佳。

新手理解重点：

1. Alpha 要和风险调整一起看。
2. 短期 Alpha 可能只是运气。
3. 基准模型选错，Alpha 也会失真。
4. 费用会吞掉真实 Alpha。

不要把高收益直接当成能力，先看它是不是靠更高风险换来的。
