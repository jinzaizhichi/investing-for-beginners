# 跨链后收到的 Token 怎么验证？

同名 Token 在不同链上可能完全不是同一种资产。

跨链前后至少核对：

1. 链 ID 和目标网络是否正确。
2. Token 合约地址是否来自官方来源。
3. Decimals 和总供应是否异常。
4. 源链是否有真实锁定或销毁记录。
5. 目标链是否有足够退出流动性。
6. 中心化交易所是否接受该版本 Token。

不要只看名称、图标和页面显示的 1 美元报价。真正重要的是：这个 Token 能不能按合理成本换回你想要的资产。

大额跨链前先做小额往返测试，并记录桥费、Gas、滑点和等待时间。
