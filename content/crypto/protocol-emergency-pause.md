# DeFi 协议 Emergency Pause 是什么？

Emergency Pause 是协议在漏洞或异常时暂停部分功能的机制。

关键是弄清楚暂停了什么：

1. 是否暂停存款。
2. 是否暂停提款。
3. 是否还能还款。
4. 是否还能清算。
5. 是否允许紧急退出。
6. 谁有权限暂停和恢复。

前端不能点，不代表链上完全不能操作；协议暂停，也不代表资产一定归零。

事件发生后，先看官方多渠道公告、链上事件和合约状态，不要点击私信里的“恢复入口”或“迁移链接”。
